#📸 Photo_Booth

click. pose. slay. repeat.

A web-based Photo Booth made with HTML, CSS & JavaScript that turns your browser into a vibe machine.
No downloads. No filters from 2012. Just clean UI and camera magic.

Built for fun. Styled for aesthetics. Powered by JS.

##🪩 What it does (aka the flex)

✨ Opens your camera in-browser
📸 Captures photos in real time
🎨 Minimal + aesthetic UI
⚡ Smooth interactions
🧠 Pure frontend logic (no frameworks, no crutches)

Basically: Instagram energy, developer edition.

🛠️ Tech Stack (aka tools of destruction)

🧱 HTML — structure, but make it cute

🎨 CSS — aesthetics doing the heavy lifting

⚙️ JavaScript — camera access & interactivity

Vanilla only. Because we’re built different.

##🎨 Design Vibes

Soft colors

Clean layout

Buttons that feel clickable

Zero clutter, max chill

If minimalism had a personality — this is it.

🚀 How to run (0 brain cells required)

Download / clone the repo

Open index.html

Allow camera access (trust the process)

Click → capture → admire 📸

That’s the whole ceremony.
