🎀✨ PHOTO BOOTH WEB APP ✨🎀
================================

📸💖 Capture moments. Save memories. Serve looks.  
A cute, minimal **web-based Photo Booth** that lets you click photos directly from your camera, add a timer, and save them instantly. Built with pure **HTML + CSS + JavaScript** — no frameworks, no drama.

---

🌸✨ FEATURES (because bare minimum is NOT our vibe)

✨ Live camera preview  
⏱ Timer-based capture (0 sec = instant slay)  
📷 Click & preview captured photos  
💾 Download photos with one tap  
🖼 Multiple photo gallery  
💅 Clean, simple, beginner-friendly code  

---

🛠 TECH STACK (soft girl edition)

🧠 HTML — structure but make it cute  
🎨 CSS — minimal, clean, aesthetic  
⚡ JavaScript — logic doing the heavy lifting  
📸 Web Camera API — the real MVP  

---

🎥 HOW IT WORKS (no overthinking bestie)

1️⃣ Allow camera access (don’t be shy)  
2️⃣ Set a timer OR leave it at 0  
3️⃣ Click the **Click** button  
4️⃣ Pose 📸  
5️⃣ Photo appears below  
6️⃣ Hit **Save** and it’s yours forever
